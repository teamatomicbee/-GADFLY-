Analog distance sensor mount for Pololu Zumo by kbuckley on Thingiverse: https://www.thingiverse.com/thing:609695

Summary:
Simple mount for Sharp analog distance sensors (specifically the 4-30cm GP2Y0A41SK0F) for Pololu's Zumo robotics platform. Mounting platform mounts to the Arduino using its mounting holes and #4-40 (or M3) hardware.
